# OOP-assignment

C++ Object-Oriented Programming assignment repository.

## Contents
- **Source code**: C++ implementation files for the assignment.
- **Build/Run**: Open the project in your preferred C++ IDE or compile with a standard compiler (e.g., `g++`).

## How to build (example)
```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic -o app *.cpp
./app
```
